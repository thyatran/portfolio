# Portfolio Website

Welcome to my personal portfolio website. This site showcases my work, skills, and projects. It was built using modern web technologies and is hosted on Netlify.

## Technologies Used

- **Frontend**: ReactJS, TailwindCSS, React Icons
- **Deployment**: Netlify
